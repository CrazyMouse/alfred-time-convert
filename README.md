# alfred-time-convert

时间格式相互转换工具，利用Alfred快速进行时间格式转换。

## Features
- 默认显示当前时间的格式化字符串以及秒和毫秒
- 对于传入参数 **2022-09-12 12:01:22** 显示秒和毫秒 
- 传入秒显示格式化时间串
- 传入毫秒显示格式化时间串
- 针对于Intel芯片和ARM芯片进行了适配

## Installation

Clone and `make dist` 


## Usage

in Alfred:

```
tt {query}
```
